问题：

监听日志，提示没有权限
mkdir: can’t create directory ‘/data/git/‘: Permission denied
此时，可能是SELINUX阻止的原因

关闭SELINUX即可解决

  • 临时关闭：setenforce 0
  • 永久关闭： vim /etc/selinux/config
